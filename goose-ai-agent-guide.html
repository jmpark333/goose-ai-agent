🤖 goose란 무엇인가?

goose는 확장 가능한 오픈소스 AI 에이전트로, 소프트웨어 개발 과정을 자동화하여 개발자의 생산성을 높여주는 도구입니다. Cursor나 GitHub Copilot과 비슷하게 AI를 활용한 코드 생성을 지원하지만, goose는 완전 무료이며 오픈소스라는 큰 장점이 있습니다.

🎯 goose 핵심 특징

  • 오픈소스: 완전 무료로 사용 가능하며, 소스 코드를 공개하여 커뮤니티 기여 가능
  • 다양한 LLM 지원: OpenAI, Anthropic, OpenRouter, Tetrate 등 다양한 프로바이더 지원
  • 확장 가능: MCP(Model Context Protocol) 서버를 통해 기능 확장 가능
  • 데스크톱 & CLI: GUI 데스크톱 앱과 터미널 CLI 버전 모두 제공
  • 자율 실행: 개발자가 최소한의 개입만으로도 복잡한 작업 자동화

goose는 Tetrate라는 AI 에이전트 라우터를 통해 $10 무료 크레딧을 제공하므로, 별도의 API 키 없이도 바로 시작할 수 있습니다. 이 점이 goose를 처음 사용하는 분들에게 매우 매력적인 요소입니다.

  • Windows: Windows 10/11 지원

설치 없이 구성만 진행하려면 다음과 같이 실행합니다.

curl -fsSL https://github.com/block/goose/releases/download/stable/download_cli.sh | CONFIGURE=false bash

goose update

🔧 권한 설정 방법

goose는 로그 디렉토리와 파일 생성을 위해 ~/.config 디렉토리에 읽기/쓰기 접근 권한이 필요합니다. 권한을 부여한 후 앱을 다시 실행하면 정상적으로 작동합니다.

goose 데스크톱 앱과 CLI는 모든 핵심 설정을 공유합니다. LLM 프로바이더 설정, 모델 선택, 확장 프로그램 설정 등이 한 번에 관리되므로, 데스크톱과 CLI를 자유롭게 전환하며 사용할 수 있습니다.

🎁 Tetrate 무료 크레딧 혜택

goose를 통해 Tetrate에 처음 인증하면 $10 무료 크레딧을 받습니다! 이 혜택은 신규 및 기존 Tetrate 사용자 모두에게 제공됩니다.

설정 단계:

  1. goose 실행 후 "Automatic setup with Tetrate Agent Router" 선택
  2. 브라우저가 자동으로 열리고 Tetrate 인증 페이지로 이동
  3. 기존 계정으로 로그인하거나 새 계정 생성
  4. 인증 완료 후 goose 앱으로 자동 복귀

Tetrate를 설정하면 여러 AI 모델을 자동으로 전환할 수 있어, 요금 제한이나 모델 장애 시에도 중단 없이 작업을 계속할 수 있습니다.

⚠️ API 키 직접 설정 시 고려사항

🌟 Z.AI GLM-4.7 설정 방법

Z.AI의 GLM-4.7는 Claude Sonnet 4.5와 경쟁하는 오픈소스 코딩 모델로, 73.8% SWE-bench 점수를 기록하며 툴 사용능력에서 87.4%를 달성합니다. goose에서 GLM-4.7을 사용하는 두 가지 방법을 소개합니다.

💎 GLM-4.7 핵심 특징

  • 355B 파라미터: 대규모 오픈소스 모델
  • MIT 라이선스: 상업적 사용 가능
  • 강력한 코딩 능력: Claude Sonnet 4.5 대등 성능
  • 툴 사용 최적화: 87.4% τ²-Bench 점수
  • 추론 유지: 대화 중 Thinking 블록 보존

방법 1: Z.AI 공식 문서 방식 (Anthropic Compatible)

Z.AI 공식 문서에 따르면 Anthropic Compatible 프로바이더를 선택하여 설정합니다.

  1. goose 데스크톱에서 Settings → Models → Configure providers
  2. Add Custom Provider 클릭
  3. Anthropic Compatible 선택
  4. Z.AI API 엔드포인트와 API 키 입력
  5. 모델 이름으로 glm-4.7 입력
// Z.AI 커스텀 프로바이더 설정 예시
{
  "name": "zai-glm",
  "engine": "anthropic",
  "display_name": "Z.AI GLM-4.7",
  "description": "Z.AI's flagship open-source coding model",
  "api_key_env": "ZAI_API_KEY",
  "base_url": "https://api.z.ai/v1/",
  "models": [
    {
      "name": "glm-4.7",
      "context_limit": 131072
    }
  ]
}

환경 변수로 API 키를 설정합니다.

# Z.AI API 키 설정
export ZAI_API_KEY="your-zai-api-key"

# goose 실행
goose

방법 2: OpenRouter 사용 (추천)

OpenRouter를 통해 z-ai/glm-4.7 모델을 사용할 수도 있습니다. OpenRouter는 여러 프로바이더에게 자동 라우팅하여 가용성을 높여줍니다.

  1. OpenRouter에서 API 키 발급
  2. goose에서 Settings → Models → Configure providers
  3. OpenRouter 선택
  4. API 키 입력
  5. 모델 선택에서 z-ai/glm-4.7 선택

CLI에서 설정하려면 다음 명령어를 실행합니다.

goose configure

메뉴에서 Configure Providers → OpenRouter를 선택하고 API 키를 입력합니다.

⚠️ GLM-4.7 사용 시 주의사항

GLM-4.7은 툴 호출 능력이 뛰어나지만, 모든 확장 프로그램이 완벽하게 호환되지 않을 수 있습니다. 특정 확장에서 문제가 발생하면 Claude 모델로 전환해 보세요.

프로바이더 선택은 예산, 사용 목적, 기술적 익숙도에 따라 결정이 달라질 수 있습니다. 무료로 시작하고 싶다면 Tetrate를, 오픈소스 고성능 모델을 원한다면 Z.AI GLM-4.7을, 특정 상용 모델을 사용해야 한다면 해당 프로바이더를 직접 설정하세요.

🖥️ Computer Controller 확장

가장 유용한 확장 중 하나는 Computer Controller입니다. 이 확장을 활성화하면 goose가 웹 브라우저를 자동으로 제어할 수 있습니다.

🎮 Computer Controller 기능

  • 웹스크래핑: 웹사이트에서 자동으로 데이터 수집
  • 파일 캐싱: 다운로드한 파일 자동 관리
  • 자동화: 반복 작업 자동 실행
  • 브라우저 제어: 웹페이지 자동 탐색 및 상호작용

활성화 단계:

  1. 왼쪽 상단 버튼으로 사이드바 열기
  2. "Extensions" 메뉴 클릭
  3. "Computer Controller" 토글로 활성화
  4. 세션으로 복귀하여 기능 사용

활성화 후 다음과 같은 프롬프트를 사용할 수 있습니다.

이처럼 goose는 단순 코드 생성을 넘어 실행, 테스트, 수정까지 자동화할 수 있습니다. 이 기능을 활용하면 개발 시간을 획기적으로 단축할 수 있습니다.

물론 goose만으로 모든 작업을 자동화할 수 있는 것은 아닙니다. 복잡한 비즈니스 로직이나 도메인 지식이 필요한 작업은 개발자와 goose의 협업이 필수적입니다. 도구 선택은 본인의 프로젝트 복잡도와 자동화 필요성에 따라 판단해야 할 지점입니다.

이 포스팅은 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다.

🎯 마치며

오늘은 goose AI 코딩 에이전트에 대해 완벽하게 알아보았습니다. goose는 무료 오픈소스라는 점, $10 무료 크레딧을 제공한다는 점, 그리고 확장 가능한 아키텍처를 갖추고 있다는 점에서 매우 매력적인 도구입니다.

초보자는 Tetrate 자동 설정으로 쉽게 시작할 수 있고, 숙련자는 다양한 확장 프로그램과 프로바이더를 활용하여 고급 자동화를 구축할 수 있습니다.

다만 goose가 모든 개발 작업을 대신해 줄 것이라 기대하기는 어렵습니다. 복잡한 문제 해결, 아키텍처 설계, 비즈니스 로직 구현은 여전히 개발자의 역량이 필요합니다. goose는 강력한 도우미일 뿐, 개발자를 완전히 대체할 수는 없습니다.

그럼에도 불구하고 생산성 향상, 반복 작업 자동화, 빠른 프로토타이핑 측면에서 goose는 분명 가치 있는 도구입니다. 여러분도 goose를 한번 사용해 보시길 추천드리면서, 저는 다음 시간에 더 유익한 정보를 가지고 다시 찾아뵙겠습니다. 감사합니다!
